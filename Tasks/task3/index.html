    • Metodology: To make the analysis of the robot easier, we can rewrite (sketch) each joint and link.

    • C谩lculo de la Matriz Final: After getting the DH parameters table and the matrix of each link, we can compute the final matrix. For this step, we use a MATLAB Live Script to simplify the calculations.

    T_final = eye(4);

     0,        0,                0,               1];

T_final = simplify(T_final * A);

    end
